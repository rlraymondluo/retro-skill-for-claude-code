# Privacy Policy

**retro** is a Claude Code plugin that runs entirely within your local Claude Code session.

## Data Collection

This plugin does **not** collect, transmit, or store any data outside your local machine. It has no network component, no telemetry, no analytics, and no external API calls.

## What the Plugin Accesses

During a session, the plugin:

- **Reads conversation history** to identify stray processes and temp files created during the session
- **Reads your Claude Code memory files** to deduplicate learnings before saving new ones
- **Writes to your Claude Code memory files** when you approve a proposed learning

All of this happens locally within your Claude Code environment. No data leaves your machine.

## Third-Party Services

This plugin does not integrate with or send data to any third-party services.
